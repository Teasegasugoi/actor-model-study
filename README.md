# actor-model-study
Pykka でアクターモデルについて学習

## 準備
```bash
$ cd actor-model-study
$ docker-compose build
```

## 起動
```bash
$ docker-compose up -d
```

## 実行
```bash
$ docker exec -it actor-model-pykka python3 xxx.py
# または
$ docker-compose exec python3 bash
[root@xxxxxxx:~/src] python3 xxx.py
```
